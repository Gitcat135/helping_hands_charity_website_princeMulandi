# CarProtect Insurance Website

## What this Website is about

- This website is about a fictional car insurance company
## What is contains

- This website has 6 different pages **They include:**
1. A home page
2. An about page
3. A programs page
4. A sign up page
5. An application page
6. And a contacts page

## Home page

- The home page has a background image with a short text stating what the website is about.
- Boxes containing bussines facts together with a short summary of the about and the programs page, and finally the footer with contains buttons to the programs and sign up page and some contact infomation.

## About page

- The about page has a short text that explains how to bussines was made, it also has 2 boxes that explain the bussinesess mission and their vission. Under that is 3 boxes that contain the bussinesess values of integrity, Customer focus and exellence in their work. And finally it contains a footer about contact information.

## Programs page

- This page contains the boxes and 3 images that explain what the bussines offers customers together with a footer identical to the about page.

## Sign up page

- This page contains a card witch has sign up information for customers to fill in. They include a text boxes for customers to fill in their name, email and payment information. Below it is the footer identical to the last 2 pages.

## Contacts page

- Lastly, this page includes 2 boxes, in one of the boxes, customers are able to fill in their email, select the subject of the question they want to ask and the message they wish to send. However, they other box contains contact information related to the bussiness.

- Under the boxes, their is a map that shows the physical location of the bussines. After that, their is a footer that contains a title asking "Interested in a job? (lawyers)" followed by a button saying, click here, if clicked the user is taken to a seperate page that has a box that contains information for users interested in a job to fill in. after that (still in the footer) is the same contact information and the other 2 pages.
 
## Technology used
- HTML to display content.
- CSS to make the website look nice.
- Bootstrap to make the page responsive and modern.
- GitHub for version control.
