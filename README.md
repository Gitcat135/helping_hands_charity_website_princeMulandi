# Helping Hands Charity Website

![Helping Hands Charity Website](./repo_images/helping_hands_charity_website.png)

This is a student-built website project. It’s a multi-page responsive site designed for a fictional charity organization called **Helping Hands**. The website is a product of mastery of HTML, CSS, Bootstrap and GitHub. Using your mastery of creating websites, you are required to create a chartiy website (it can be on anything - what we have here is about education).

## Features

- Nice landing page with a hero background image
- Information **about** Helping Hands Charity organization
- Some of the programs being done by the organization
- A call to donate so that the work can continue
- A contact page for ease of access

## Technology Used

- HTML to display content
- CSS to make the website look nice
- Bootstrap to make the page responsive and modern
- GitHub for version control
- Netlify for deployment

## Instructor Notes

This project is part of a hands-on class to help students practice full-site layout using real-world tools. It can be extended with forms, donations integrations, or accessibility improvements.
